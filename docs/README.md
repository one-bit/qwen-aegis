# AEGIS Framework Reference

This directory contains the original AEGIS framework specifications, adapted for Qwen Code.

- `standards/` — Design standards and conventions for agents, commands, domains, personas, rules, schemas, tools, and workflows.
- `validation/` — Convention compliance reports, cross-reference reports, and validation summaries.

For usage documentation, see the extension's main `README.md` at `~/.qwen/extensions/aegis/README.md`.

For the original AEGIS project, see https://github.com/ChristopherKahler/aegis.
